# transport-front-end
This app displays train details to set locations.

## Installation
`yarn install`

## Tests
`yarn test`

## Push to CI/QA
`git push`

## Push to production
`git push heroku master`
OR
Navigate to 'Deploy' tab on Heroku scroll to 'Manual Deploy' section and click deploy branch.
